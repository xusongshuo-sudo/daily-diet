健脾补肾补气血 一周饮食卡

饮食原则

调脾胃、补气血、补肾气三目标

多吃：虾仁、山药、玉米、生菜、南瓜、番薯

早餐新增：黄豆豆浆、黑豆豆浆、全麦吐司

忌：生冷、油炸、奶茶、宵夜、冰饮

喝汤当天：清淡、不喝咖啡浓茶

一周调理饮食计划

周一 健脾调理

早餐

黄豆豆浆 + 小米南瓜粥 + 水煮蛋

午餐

杂粮饭 + 清炒山药 + 虾仁生菜

下午

红枣3颗

晚餐

山药玉米排骨汤 + 清炒菠菜

周二 补气血调理

早餐

红豆红枣粥 + 全麦吐司1片

午餐

玉米饭 + 西兰花炒虾仁

下午

当归北芪桂圆红枣鸽子汤

晚餐

鲫鱼豆腐汤 + 生菜

周三 补肾气调理

早餐

黑豆豆浆 + 三黑粥 + 蒸南瓜

午餐

杂粮饭 + 板栗排骨 + 清炒山药

下午

核桃/腰果一小把

晚餐

山药枸杞瘦肉汤 + 生菜

周四 脾胃休整

早餐

黄豆豆浆 + 山药莲子粥 + 水煮蛋

午餐

荞麦饭 + 清蒸鱼 + 生菜

下午

蒸番薯一小块

晚餐

小米粥 + 玉米 + 豆腐干

周五 暖肾调理

早餐

黑豆豆浆 + 红枣枸杞小米粥

午餐

清蒸驴肉

+ 玉米 + 生菜

下午

原味花生

晚餐

猪肝菠菜粥（少量）

周六 综合滋补

早餐

黄豆豆浆 + 莲子芡实粥 + 蒸番薯

午餐

杂粮饭 + 黄焖鸡 + 清炒山药

下午

肝肾脾胃综合滋补汤（15-17点）

晚餐

菌菇汤 + 生菜 + 玉米

滋补汤配方：

熟地15g 山药20g 当归10g 黄芪15g 枸杞10g 莲子10g 黑芝麻10g 红枣5颗 生姜3片

做法：泡30分钟 → 加水1500ml → 大火煮开小火炖1小时

可加：鸡腿/猪腱子肉｜每周1-2次

周日 清淡调理修复

早餐

黑豆豆浆 + 全麦吐司1片 + 蒸南瓜

午餐

杂蔬鸡肉焖饭（加玉米）

下午

陈皮茯苓水

晚餐

枸杞乌鸡汤 + 生菜沙拉

经后专属调理

月经干净后 连喝4-5天

补气血不胖

配方：

桂圆5-10颗 + 当归10-15g + 大枣5颗 + 枸杞15-20g

功效：

  • 暖宫寒
  • 补气血
  • 调量少
  • 止痛经
